To Use:

1: git clone https://github.com/Rueote/FieldNotes.git

2: cd FieldNotes

3: npm install

4: npm run dev

**Getting Started**:

Open Command Prompt

Press the Start button

Type cmd

Click Command Prompt

Copy and paste this, then press Enter:

git clone https://github.com/Rueote/FieldNotes.git

Go into the project folder:

cd FieldNotes

Install everything the project needs:

npm install

Start the app:

npm run dev

When it finishes, it will show a link like:

http://localhost:5173

Open that in your browser.



**About**

FieldNotes is a script editor that lets you write your screenplay, add tags, and break down scenes in a simple and organized way.
